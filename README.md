# 💻 Hagaha Tiknoolajiyada IT-ga ee Af-Soomaaliga

Dukumiintigani wuxuu ururinayaa qalabyo, teknoolojiyo, qaabab xogeed, iyo fikrado muhiim u ah horumarinta, amniga, shabakadaha, daruuraha, DevOps, iyo maaraynta nidaamyada. Waxaa loogu talagalay in lagu taageero ardayda Soomaaliyeed ee baranaya shahaadooyinka IT-ga sida Microsoft Azure, Linux, iyo ITIL 4.
